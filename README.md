# Student Management System

## 📌 Description
This is a Spring Boot project for managing student data using CRUD operations.

## 🚀 Features
- Add Student
- View Student
- Update Student
- Delete Student

## 🛠 Technologies Used
- Java
- Spring Boot
- MySQL
- Hibernate (JPA)
- Postman

## ⚙️ How to Run
1. Run the Spring Boot application
2. Use Postman to test APIs
3. Connect with MySQL database

## 📬 API Endpoints
- POST /students → Add student
- GET /students → Get all students
- PUT /students/{id} → Update student
- DELETE /students/{id} → Delete student
